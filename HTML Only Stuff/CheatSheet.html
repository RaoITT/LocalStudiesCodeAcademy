HTML CheatSheet

HTML
Tags Attribute
Tag Descritpion Attribute Descritpion
table The wrapper element for all HTML tables colspan Defines how many columns a td element should span.
tr Sets off each row in a table rowspan Defines how many rows a td element should span
td Sets off the data of the table border Sets the width of the border around table cells
thead The set of rows defining the column headers in a table. cellspacing Sets amount of space between table cells
tbody The set of rows containing actual table data. cellpadding Sets amount of space between a cell's border and its contents
tfoot The set of rows defining the footer in a table. valign Sets vertical alignment for cells within the row(top/middle/bottom)
html Creates an HTML document align Sets alignment for cells within the row(left/center/right)
head Sets off the title & other info that isn't displayed nowrap Prevents lines within a cell from being broken to fit
body Sets off the visible portion of the document
pre Creates preformatted text
hx(1>= x >= 6) Creates headlines -- H1=largest, H6=smallest
b Creates bold text
em Emphasizes a word (usually processed in italics)
strong Emphasizes a word (usually processed in bold)
a Creates a hyperlink to a Uniform Resource Locator
form Defines a form
CSS
Property Value
Color Changes the color of text.
Background-Color Sets the background color of an element.
Background-Image Sets the background image of an element.
Font-Family Changes the font family of certain words, sentences,
Font-Style Changes text: normal, oblique, and italics.
Font Used to combine all properties of fonts.
Font-Size Used to modify the size of the displayed font.
Font-Weight Used to specify the weight of the font.
Text-Align Used to justify text left, center, right, and justify.
Padding Shorthand for the padding-top, padding-right, padding-bottom, and padding-left properties.
Float Allows text to wrap around an element (left, right, none).
Clear Specifies whether an element allows floating elements to its sides (left, right, none).
